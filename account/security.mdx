# Security

## Platform Security

All data on Qualtranscribe is protected by the following technical controls:

- **Encryption at rest:** 256-bit AES on all stored files and data
- **Encryption in transit:** TLS 1.3 on all data transmission
- **Multi-factor authentication:** Available for account access
- **Role-based access controls:** Personnel access only what they need for a specific project
- **Automatic session timeouts:** Active sessions expire after a period of inactivity
- **Comprehensive audit logging:** All PHI access and system activities are logged with user ID, timestamp, and action type
- **Secure file storage:** Wasabi Technologies S3-compatible object storage with regional data residency

## Data Residency

Your data residency region is set when you create your account.

| Region | Storage Location |
|---|---|
| United States | Wasabi us-east-1, North Virginia |
| European Union | Wasabi eu-central-2, Frankfurt, Germany |

EU accounts store all data within the EEA. AI processing for EU accounts (Instant Draft via AssemblyAI and Smart Insights via AWS Bedrock) also occurs within EU infrastructure.

## Account Password

Your password is hashed and never stored in plain text. To change your password:

1. Go to **Settings** in the sidebar
2. Select the **Security** tab
3. Enter your current password and new password
4. Save

## Forgotten Password

On the login page, click **Forgot password** and enter your email address. A reset link will be sent within a few minutes.

## US-Only Processing

For clients with institutional requirements, IRB restrictions, federal funding mandates, or legal requirements prohibiting international data transfer, US-Only processing is available. When this option is active:

- All transcription is performed exclusively by US-based professionals
- No international data transfer occurs for any part of the workflow
- All file storage and any AI processing remain within the United States

Specify US-Only processing when placing your order, or contact support to configure it as the default for your account.

## Operational Security

**Staff and contractor controls:**
- All transcriptionists and contractors sign Non-Disclosure Agreements before accessing any client files
- Access to files is time-limited and project-specific; access is automatically revoked upon project completion
- Mandatory HIPAA training for all staff handling healthcare-related content
- Encrypted, password-protected workstations required
- Prohibition on accessing client data in public locations or on shared devices

**Incident response:**
- For GDPR projects: supervisory authority notified within 72 hours of a confirmed breach, affected clients notified without undue delay
- For HIPAA projects: covered entity notified within 24 hours of a confirmed breach
- Certificate of Destruction available upon request for deleted projects

## Reporting a Security Concern

Contact support@qualtranscribe.com immediately if you believe your account has been compromised or if you identify a potential security issue. We treat all security reports as high priority.
