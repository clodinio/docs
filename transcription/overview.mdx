# Transcription Overview

Qualtranscribe offers two transcription services. Both accept the same file formats and integrate with the same cloud storage providers — the difference is in who (or what) does the transcribing, and how quickly you receive the result.

## Human Transcription

Professional transcriptionists handle your file from start to finish. This is the right choice when:

- You need the highest possible accuracy
- Your recording has multiple speakers, heavy accents, or domain-specific terminology
- You need verbatim formatting for qualitative analysis software
- You're preparing transcripts for publication or academic submission

## Instant Draft

AI-powered transcription delivered in minutes. This is the right choice when:

- You need a quick first pass before deeper analysis
- Turnaround speed matters more than perfection
- You're working with clear, single-speaker recordings
- You want to review content before commissioning a human transcript

## Choosing the right service

| | Human Transcription | Instant Draft |
|---|---|---|
| Accuracy | Highest | Good |
| Turnaround | Hours to days | Minutes |
| Speaker labels | ✅ Add-on | ✅ Automatic |
| Verbatim options | ✅ Full or Clean | Clean only |
| Timestamps | ✅ Every 2 mins | ✅ Automatic |
| Translation | ✅ Add-on | — |
| Best for | Publication, analysis | Quick review |

<Tip>
  Not sure which to use? Start with an Instant Draft to review the content, then order a Human Transcription for final quality.
</Tip>
