# Managing Files

## Uploading Recordings

Within any WorkHub project or folder, you can upload audio and video recordings directly. Files are stored securely and associated with the folder they were uploaded into.

1. Open the relevant project or folder
2. Click **Upload**
3. Select your file or drag and drop it into the upload zone
4. The file appears in the folder and is ready for a transcription order

## Viewing Transcripts

Completed transcripts are returned directly to the folder where the original recording is stored. Transcripts can be viewed with Microsoft 365 Online Viewer or downloaded on your device. HIPAA-active accounts can only download transcripts and cannot view documents on Microsoft 365 Online Viewer.

## File Naming

Use consistent, descriptive file names before uploading. Good naming conventions make it easier to locate recordings, match recordings to transcripts, and import files into qualitative analysis software.

A common convention for research projects: `[ProjectCode]_[ParticipantID]_[Date]_[SessionType].mp3`

For legal projects: `[MatterNumber]_[Witness]_[Date]_Deposition.mp4`

## Retention

File retention is configurable from your account settings under **HIPAA & Compliance**. Available retention periods:

| Period | Description |
| --- | --- |
| 30 days | Default |
| 60 days |  |
| 90 days |  |
| 180 days |  |
| 1 year |  |
| Never | Files are retained until manually deleted |

Download or archive your transcripts before your selected retention period expires. Files deleted at the end of the retention period are permanently removed and cannot be recovered.

<Note>
  For projects with ongoing activity, make sure your retention policy is set to cover the full anticipated project period before uploading files. Update your retention setting in **Settings under HIPAA & Compliance**.
</Note>
